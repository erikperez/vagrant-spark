# vagrant-spark

## Prerequisites

* [Oracle VirtualBox](https://www.virtualbox.org)
* [Vagrant](https://vagrantup.com)

##Installation

* Open console and type ```vagrant up```


### What you get
1. ```Ubuntu 14.04 Lts```
2. ```Git```
3. ```Oracle Java 8```
4. ```Scala 2.11.7```
5. ```Spark 1.6.0 for Hadoop 2.6 (prebuilt)```

More info will come.
